# Site Prototype

Пока это статическая локальная страница без backend.

Что сейчас есть:

- простая регистрация
- имя персонажа
- email
- пароль
- сохранение данных в `localStorage`

Что сейчас убрано:

- редактор персонажа
- выбор окраса
- возраст

Как открыть локально:

## Вариант 1. Просто открыть файл

Откройте:

- `site/index.html`

## Вариант 2. Через локальный сервер

Если установлен Python:

```bash
cd site
python -m http.server 4173
```

Потом открыть в браузере:

- `http://localhost:4173`

Данные сейчас сохраняются только локально в браузере, по ключу:

- `fanglaw.site.prototype.account`

Следующий этап:

- подключить настоящий backend
- заменить `localStorage` на API
- после регистрации добавить отдельный экран создания персонажа
