# Walking-ассеты персонажа

Эта папка подготовлена под отдельную walking-модель для сайта.

Здесь должны лежать только walking-специфичные body-слои.

## Структура

- `body/base/`
- `body/shadows/`
- `body/contours/`
- `patterns/body/`

## Что сюда класть

### Обязательные базовые слои

- `body/base/bodybase.png`
- `body/shadows/bodybaseshadow.png`
- `body/contours/bodybaseco.png`

### Body-паттерны

Если паттерн должен подстраиваться под walking-лапы или живот, его копия должна лежать здесь:

- `patterns/body/pattern1.png`
- `patterns/body/pattern2.png`
- ...
- `patterns/body/pattern23.png`

### Дополнительно, если потребуется

Если для walking-позы нужно отдельно подогнать маски лица, можно тоже положить сюда:

- `patterns/body/eyeswhite.png`
- `patterns/body/pattern12nose.png`

## Именование walking-паттернов

- Если walking-версия совпадает со старой, можно положить файл под тем же именем:
  `patterns/body/pattern1.png`
- Если walking-версия отличается от обычной, используется тот же номер, но с суффиксом `idet`:
  `patterns/body/pattern17idet.png`
- `pattern17idet` считается walking-вариантом для обычного `pattern17`.

## Что не нужно складывать сюда

Пока не нужно дублировать:

- уши
- хвосты
- tail patterns
- гриву
- щёки

Они остаются в обычных папках, пока не появится реальная необходимость сделать для них walking-вариант.

## Как это использовать

- Сайт не должен просить игрока собирать walking-вариант отдельно.
- Основная внешность персонажа выбирается один раз.
- Walking-файлы нужны как отдельный набор body-ассетов, который игра может подхватывать автоматически во время движения.
