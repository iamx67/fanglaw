# Ассеты редактора персонажа

Текущее правило простое:

- белым или светло-серым рисуем только то, что потом будет краситься;
- черным рисуем контур, зрачки и прочие детали, которые не должны менять цвет;
- фон у всех PNG прозрачный.

Что класть по папкам:

- `body/`
  - базовое тело персонажа;
  - черный контур + белая заливка шерсти;
  - размер для первого MVP лучше держать `128x128`.

- `eyes/`
  - `eyes_base`:
    - черный контур, белки, блики, все что не должно перекрашиваться;
  - `eyes_iris`:
    - белая радужка, которая будет краситься в выбранный цвет глаз.

- `patterns/`
  - узоры шерсти;
  - отдельные PNG на прозрачном фоне;
  - белая заливка, чтобы их можно было красить отдельно.

- `accessories/`
  - ошейники, украшения, элементы одежды и т.п.;
  - для первого этапа можно рисовать их уже готовыми по цвету.

Что лучше не делать:

- не рисовать перекрашиваемые части черными;
- не делать много одинаковых файлов только ради разных цветов;
- не смешивать в одном слое и перекрашиваемую часть, и статичные блики/белки глаз, если нужен чистый цвет.

Минимальный набор для старта:

1. `body/body_base.png`
2. `eyes/eyes_base.png`
3. `eyes/eyes_iris.png`
4. `patterns/pattern_01.png`

Этого уже хватит, чтобы собрать первый редактор персонажа.
