# Web export для `fanglaw`

В эту папку нужно складывать готовый web-export Godot-клиента перед загрузкой на VPS.

Ожидаемый результат после экспорта:

- `index.html`
- `*.js`
- `*.wasm`
- `*.pck`

На VPS эти файлы будут лежать в:

```text
/var/www/fanglaw-web/current
```

Важно:

- для web-сборки текущий клиент должен оставаться на `GDScript`;
- Godot 4 `.NET/C#` в web не экспортируется;
- в браузере клиент автоматически подключается к тому же домену, где открыт сайт.
